# shopping-cart-react

The project contains three features, which implement with React:
1. collapsible add to shopping cart menu
2. reactive shopping cart badge


![image](https://github.com/EnChiSu/shopping-cart-react/blob/main/Demo.gif)


## Run Locally
Clone the project
```
git clone https://github.com/EnChiSu/shopping-cart-react.git
```

Go to the project directory
```
cd shopping-cart-react
```

Install dependencies
```
npm install
```

Start the server
```
npm start
```
